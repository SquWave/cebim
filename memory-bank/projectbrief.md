# Project Brief: Cebim (Personal Finance & Investment Tracker)

## Core Requirements
- **Unified Dashboard**: View budget status and investment portfolio in one place.
- **Expense Tracking**: Quick entry for daily income/expenses, categorize spending, monthly budget tracking.
- **Investment Tracking**: Manual entry of investment assets, real-time total portfolio value updates (manual price update for MVP).
- **Local-First Security**: All data stored securely on the device using localStorage.
- **Simplicity**: Simple but powerful tool, starting with core functionality.
- **Platform**: PWA (Progressive Web App) optimized for mobile and desktop.

## Goals
- Create a functional MVP that combines finance and investment tracking.
- Ensure data privacy with local storage.
- Provide a seamless user experience for daily use with a "Dark Mode" first design.
- Build a responsive Single Page Application (SPA) using React and Tailwind CSS.
